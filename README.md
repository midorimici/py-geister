# py-geister

Python と pygame によるガイスター。

[作成過程](https://midorimici.com/posts/geister-app-dev-1/)

<br>

# 必要なもの

python 3.8.3 と pygame 1.9.6 で動作を確認しています。

pygame のインストールは

```
pip install pygame
```

<br>

# 起動

通常起動

```
py main.py
```

フルスクリーン起動

```
py main.py f
```

<br>

# 音声素材について

このゲームの効果音素材は[On-Jin ～音人～](https://on-jin.com/)のものを使用しています。
